¿Qué es un repositorio remoto en GitHub?
Un repositorio remoto es una ubicación en GitHub donde los códigos fuente de un proyecto se almacenan de forma centralizada. Ayuda a muchos desarrolladores a trabajar en un único proyecto en ubicaciones geográficamente dispersas y a mantener las versiones.

Para trabajar con Git y GitHub, inicialmente, tienes un repositorio local en tu ordenador y luego utilizas el de GitHub para actualizar tu trabajo.

Para crear un nuevo repositorio en GitHub, sólo tienes que seguir estos sencillos pasos:
Inicia sesión en GitHub

Ve a https://github.com e inicia sesión.
Crea un nuevo repositorio

Haz clic en el botón superior derecho «＋» y luego en «Nuevo repositorio».
Crea un repositorio Git

Introduce un nombre para el repositorio
Ocasionalmente, añada una descripción
Elijo transparencia
Público: es accesible a todo el mundo

Privado: accesible sólo para el propietario y los donantes

No elija inicializar con un README.md, un.gitignore, o una licencia (esto se hará a través de Git). Crear un repositorio Haz clic en «Crear repositorio». GitHub presentará una página con un tutorial para mapear un repositorio remoto en uno local.
